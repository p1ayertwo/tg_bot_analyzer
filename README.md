## Описание:
Этот Telegram-бот предоставляет актуальные цены активов и показывает динамику за последний день. Он написан на Python, интегрирован с Yahoo Finance и Московской Биржей.
## Инструкция:
1. Создайте нового бота в Telegram через [@BotFather](https://t.me/botfather) и получите токен.
2. Создайте файл `.env` и добавьте токен вашего бота:
   ```dotenv
   TELEGRAM_API = <токен>
   ```
3. Запустите бота:
   ```bash
   python main.py
   ```
4. Отправьте команду `/start`, чтобы начать разговор с ботом.
